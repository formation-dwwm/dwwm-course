Environnement PHP

PHP est un langage serveur. C'est un langage de script et a donc besoin d'un moteur pour être exécuté. Le langage PHP vient avec un utilitaire, php, notamment capable d'exécuter les fichiers utilisant ce langage. Il est parfois utilisé pour créer des outils en ligne de commande, et le plus souvent utilisé pour réaliser des sites internet. Du point de vue PHP, ces derniers peuvent être mis en oeuvre directement via la génération de pages dynamiques (création de code html), ou la mise à disposition d'APIs (communication le plus souvent en JSON).
Sur un serveur web, ce n'est pas PHP qui fera office de serveur http (et qui écoutera donc les requêtes). PHP a besoin d'être accompagné pour cette tâche, et le plus souvent ce sera Apache ou Nginx qui aura ce rôle. Et étant très souvent utilisé avec un SGBD relationnel, notamment MySQL, on parle du stack LAMP pour Linux, Apache, MySQL et PHP qui représente une part importante des serveurs web.
Pour vos développements locaux, il vous faudra donc un équivalent. Pour simplifier l'installation et configuration de ces différents logiciels, nous vous conseillons d'utiliser Wamp ou Xamp que vous devriez déjà connaître.
