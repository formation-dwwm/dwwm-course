Pensez à réserver un temps pour retravailler la clareté de votre code.

# Stack

Choix entièrement libre. Génération des pages sur serveur et/ou utilisation AJAX... Tout vous est permis.

# Livrables

- MCD, MLD, MPD
- Votre dossier de projet compressé
- Un export de votre base de données et/ou un script d'initialisation

# Bonus

Implémenter les fonctionnalités en orange sur les maquettes.
